Wichtige Linux-Befehle

Die folgenden Befehle sind in der Eingabeaufforderung einzugeben.

SSH-Verbindung mit einem Server herstellen:
ssh benutzername@server-adresse
Beispiel: ssh typo3@192.168.54.240

Die Server-Adresse kann eine IP-Adresse oder ein Domainname sein. Anschließend wird man zur Eingabe des SSH-Passwortes aufgefordert (für die im Training verwendete virtuelle Maschine: typo3)
Dateien herunterladen:
wget Zieldatei
Beispiel: wget http://prdownloads.sourceforge.net/typo3/typo3_src-7.6.2.tar.gz
Archive auspacken:
tar xzf dateiname.tar.gz
Beispiel: tar xzf typo3_src-7.6.2.tar.gz
Archiv erstellen:
tar czf dateiname.tar.gz verzeichnisname/
Beispiel: tar czf typo3projekt1.tar.gz typo3projekt1/
MySQL-Datenbank exportieren:
mysqldump --opt -u datenbankbenutzer -ppasswort datenbankname > datenbankexportdatei.sql
Beispiel: mysqldump --opt -u root -ptypo3 typo3projekt1 > db_typo3projek1.sql

Je nach Serverumgebung muss hier der Datenbank-Server mit dem Parameter -h angegeben werden.
Beispiel: mysqldump --opt -h 127.0.0.1 -u root -ptypo3 typo3projekt1 > db_typo3projek1.sql
MySQL-Datenbank importieren:
mysql -u datenbankbenutzer -ppasswort datenbankname < datenbankexportdatei.sql
Beispiel: mysql -u root -ptypo3 typo3projekt2 < db_typo3projekt1.sql

Je nach Serverumgebung muss hier der Datenbank-Server mit dem Parameter -h angegeben werden, z.B.
Beispiel: mysql -h 127.0.0.1 datenbankbenutzer -ppasswort datenbankname < datenbankexportdatei.sql
Verzeichnis anlegen:
mkdir Verzeichnisname
In ein Verzeichnis wechseln:
chdir Verzeichnisname
Eine Verzeichnisebene nach oben wechseln:
chdir ..
Verzeichnis-Inhalt anzeigen:
ll (oder ls -al)
Benutzer/Benutzergruppen für Verzeichnisse oder Dateien rekursiv ändern:
chown -R benutzername:benutzergruppe verzeichnis/
Beispiel: chown -R typo3:www-data typo3projekt1/

Der Parameter -R steht für "rekursiv", d.h., auch Unterverzeichnisse des Zeilverzeichnisses werden bearbeitet.
(Weitere Dokumentation im Netz unter der URL: https://wiki.ubuntuusers.de/chown/ )
Lese-/Schreibberechtigungen für Dateien/Verzeichnisse ändern:
chmod optionen berechtigungen verzeichnis/

Da sehr viele Kombinationen mit diesem Befehl möglich sind, verweisen wir hier auf eine Dokumentation: https://wiki.ubuntuusers.de/chmod/
Texteditor nano:
Editor starten: nano dateiname

Datei speichern:<Strg> + <o> (Mac: <Ctrl> + <o>)
Editor beenden: <Strg> + <>x (Mac: <Ctr>l + <x>)
Datei-Manager Midnight Commander:
Midnight Commander starten: mc

Midnight Commander beenden: <F10>
Symlink editieren: <Strg> +<x> + <s> (Mac: <Ctrl> + <x> + <s>)
